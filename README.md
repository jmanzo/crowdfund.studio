# Crowdfund Studio - Shopify Crowdfund Studio App' Landing Page.

## How to use

### 1\. Clone this Repository

```bash
git clone https://github.com/surjithctly/nextly-template.git
```

### 2\. Navigate to the directory

```
cd crowdfund.studio
```

### 3\. Install dependencies

```
yarn install
# or
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
